# 學習歷程自述 AI 教練

![學習歷程自述 AI 教練簡介圖](簡介圖.png)

這是一份可放入 Google Antigravity 2.0 專案根目錄的 `AGENTS.md`，用來把 AI 代理人設定成「學習歷程自述教練」。

它的定位不是代寫工具，而是協助高中生透過多輪對話盤點經驗、整理反思、連結目標校系，最後完成保留個人真實聲音的學習歷程自述。

## 專案特色

- 以 `AGENTS.md` 作為單一自足的專案規則檔，不需要額外安裝 skill 或子檔案。
- 啟動後依序走五階段：盤點經驗、回顧歷程、整理亮點、連結未來、表達修整。
- 內建不代寫、不編造、不誇大、追問空泛內容、建議教師審閱等治理原則。
- 適用於學習歷程、自述、備審資料、多元表現、就讀動機、學習計畫等準備情境。
- 內含 12 個常見學群的審查重點速查表，協助學生把真實經驗對應到校系需求。

## 適合誰使用

- 正在準備大學個人申請、推甄或備審資料的高中生。
- 想協助學生整理學習歷程，但希望保留學生主體性的教師或輔導人員。
- 想在 Antigravity 專案中建立穩定教練流程的教學者。

## 快速開始

1. 下載或複製本專案。
2. 在 Google Antigravity 2.0 建立一個新專案。
3. 將本專案的 `AGENTS.md` 放在 Antigravity 專案資料夾的最外層。
4. 開啟新的對話，直接用白話描述需求，例如：
   - `我想申請資工系，幫我準備備審資料`
   - `幫我寫學習歷程自述`
   - `我參加過科展，但不知道怎麼寫`
5. 教練會從經驗盤點開始追問細節，並協助你逐步整理素材。

確認是否生效時，可以在 Antigravity 中問：

```text
你現在的角色是什麼？要遵守哪些規則？
```

如果回覆提到學習歷程自述教練、五階段流程，以及不代寫、不編造等原則，就表示設定已生效。

## 五階段流程

| 階段 | 目標 | 主要產出 |
|------|------|----------|
| 1. 盤點經驗 | 把模糊記憶整理成具體素材 | 經驗素材清單 |
| 2. 回顧歷程 | 找出經驗間的共通點、成長脈絡或轉折 | 成長脈絡 |
| 3. 整理亮點 | 對照目標校系看重的能力 | 能力亮點對照表 |
| 4. 連結未來 | 串起過去經驗、現在選擇與未來計畫 | 過去到未來的敘事鏈 |
| 5. 表達修整 | 協助學生修改自己寫出的初稿 | 自述文章修改建議 |

## 核心原則

- 不代寫：只提供結構、提問與修改方向，不產出整段可直接交出的原創文章。
- 不編造：所有經驗細節都必須來自學生自述。
- 不誇大：保持真實比例，不把普通經驗包裝成過度戲劇化的轉折。
- 可追溯：修改建議要保留原始版本，讓學生能比較前後差異。
- 教師把關：最終版本建議交由指導教師審閱。
- 追問空泛：遇到「我學到很多」「收穫很大」等說法時，會追問具體例子。

## 使用提醒

這個教練適合用來引導整理與反思，不適合用來取代學生自己的書寫。完成自述後，請交由指導教師或輔導老師審閱，確認內容真實、語氣自然，也符合目標校系要求。
